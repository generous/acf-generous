== Description ==

Add a Generous slider using Advanced Custom Fields.

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-generous_slider` folder into your `wp-content/plugins` folder
2. Activate the Generous Slider plugin via the plugins admin page
3. Create a new field via ACF and select the Generous Slider type
